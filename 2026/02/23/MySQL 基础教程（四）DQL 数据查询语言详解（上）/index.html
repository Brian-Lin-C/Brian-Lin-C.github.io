3.1 比较运算符

运算符 说明 示例
= 等于 salary = 8000
!= 或 <> 不等于 dept_id != 1
> 大于 age > 30
< 小于 age < 30
>= 大于等于 salary >= 8000
<= 小于等于 salary <= 10000
<=> 安全等于（可比较 NULL） dept_id <=> NULL

3.2 逻辑运算符

运算符 说明 示例
AND 或 && 与 age > 25 AND salary > 8000
OR 或 || 或 dept_id = 1 OR dept_id = 2
NOT 或 ! 非 NOT gender = '男'

四、聚合函数

聚合函数对一组数据进行计算，返回单个值。

4.1 常用聚合函数

函数 功能 说明
COUNT() 计数 统计行数
SUM() 求和 计算总和
AVG() 平均值 计算平均数
MAX() 最大值 找出最大值
MIN() 最小值 找出最小值

4.2 COUNT 计数

COUNT(*) vs COUNT(列) vs COUNT(1)：

  • COUNT(*)：统计所有行（包括 NULL），推荐用于计数
  • COUNT(列)：统计该列非 NULL 的行数
  • COUNT(1)：效果同 COUNT(*)，但某些场景略慢
