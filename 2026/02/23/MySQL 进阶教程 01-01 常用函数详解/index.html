MySQL 进阶教程 01-01 常用函数概述与字符串函数

在基础教程中，我们学习了 SQL 的基本查询和操作。本文将系统讲解 MySQL 提供的常用内置函数，重点介绍字符串函数的使用方法。

一、函数概述

MySQL 函数是对数据进行特定操作的预定义程序，可以简化 SQL 语句，提高开发效率。

函数的分类：

  • 字符串函数：处理字符串数据
  • 数值函数：处理数值数据
  • 日期时间函数：处理日期和时间
  • 流程函数：条件判断和流程控制
  • 聚合函数：对一组数据进行计算（已在 DQL 章节讲解）
  • 加密函数：数据加密和哈希
  • 其他函数：信息函数、加锁函数等

三、小结

核心知识点

  1. 字符串连接：CONCAT 连接多个字符串，CONCAT_WS 用分隔符连接
  2. 字符串长度：LENGTH 返回字节数，CHAR_LENGTH 返回字符数
  3. 大小写转换：UPPER/LOWER 转换大小写
  4. 字符串截取：LEFT/RIGHT/SUBSTRING 截取子串
  5. 查找替换：INSTR/LOCATE 查找位置，REPLACE 替换字符串

注意事项

  1. MySQL 中字符串位置从 1 开始，不是从 0
  2. LENGTH 和 CHAR_LENGTH 的区别（字节 vs 字符）
  3. REPLACE 是大小写敏感的
