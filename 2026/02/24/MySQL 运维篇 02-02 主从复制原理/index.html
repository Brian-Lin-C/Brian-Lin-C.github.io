七、GTID 复制

7.1 什么是 GTID

GTID（Global Transaction Identifier）：

  • 全局唯一的事务标识符
  • 格式：source_id:transaction_id
  • 示例：3E11FA47-71CA-11E1-9E33-C80AA9429562:23

8.2 并行复制类型

MySQL 5.6：

  • 按库并行：不同库的事务可以并行
  • 限制：同一库的事务串行

MySQL 5.7+：

  • LOGICAL_CLOCK：基于组的提交
  • 同一组的事务可以并行

MySQL 8.0+：

  • WRITESET：基于写集依赖
  • 更精确的依赖判断

复制流程总结

核心组件

组件 位置 作用

SQL Thread 从库 重放操作

关键概念

  1. 异步复制：主库不等待从库
