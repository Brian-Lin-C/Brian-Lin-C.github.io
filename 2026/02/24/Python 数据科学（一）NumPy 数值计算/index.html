一、NumPy 简介

1.1 为什么使用 NumPy？

  • 性能优势：底层用 C 实现，比纯 Python 快 10-100 倍
  • 内存效率：连续存储，占用更少内存
  • 向量化操作：无需循环，代码更简洁
  • 生态基础：Pandas、SciPy、scikit-learn 等都基于 NumPy

九、性能优化建议

提醒自己：

  1. 使用向量化：避免 Python 循环
  2. 利用广播：减少内存复制
  3. 选择合适的数据类型：float32 比 float64 快且省内存
  4. 原地操作：使用 out 参数减少内存分配
  5. 使用 views 而非 copies：切片时注意
