IDE 工具配置：VS Code + Jupyter + DataGrip

工欲善其事，必先利其器。本篇将介绍三款常用的开发工具：VS Code（通用 IDE）、Jupyter Notebook（数据科学）和 DataGrip（数据库管理）。

一、VS Code

1.1 简介与安装

Visual Studio Code (VS Code) 是微软开发的免费开源代码编辑器，支持几乎所有主流编程语言。

特点：

  • 免费开源
  • 跨平台（Windows、macOS、Linux）
  • 丰富的插件生态
  • 轻量快速
  • 内置 Git 支持

下载安装：

  • 官网：https://code.visualstudio.com/
  • 下载对应系统版本安装

三、DataGrip

3.1 简介与安装

DataGrip 是 JetBrains 出品的专业数据库管理工具，支持多种数据库。

支持的数据库：

  • MySQL / MariaDB
  • PostgreSQL
  • Oracle
  • SQL Server
  • SQLite
  • MongoDB (通过插件)
  • Redis (通过插件)
  • 以及更多…

下载安装：

  • 官网：https://www.jetbrains.com/datagrip/
  • 30 天试用，之后需要订阅
  • 学生可免费使用（凭学生证）

四、工具整合

4.1 VS Code + Jupyter

在 VS Code 中运行 Jupyter Notebook：

  1. 安装 Jupyter 和 Python 插件
  2. 打开 .ipynb 文件
  3. 选择 Python 内核
  4. 运行单元格（Shift + Enter）

优势：

  • 在 VS Code 中编辑 Notebook，享受 VS Code 的所有功能
  • 支持变量查看器、数据查看器
  • 支持交互式窗口

4.2 VS Code + 数据库

在 VS Code 中管理数据库：

安装插件：

  • MySQL (Jun Han)
  • PostgreSQL (Microsoft)
  • SQLTools (Matheus Teixeira)

使用 SQLTools：

  1. 安装插件
  2. 配置连接
  3. 在侧边栏查看数据库结构
  4. 执行 SQL 查询

4.3 工作流建议

日常开发：

  • 代码编辑：VS Code
  • 数据分析：Jupyter Notebook
  • 数据库管理：DataGrip

数据科学项目：

  1. 用 DataGrip 探索和导出样本数据
  2. 用 Jupyter 进行数据探索和建模
  3. 用 VS Code 编写生产代码
  4. 用 Git 进行版本控制

五、最佳实践

提醒自己：

  1. 善用快捷键：提高效率，减少鼠标使用
  2. 配置同步：使用 Settings Sync 同步配置
  3. 插件适度：不要安装太多插件，影响性能
  4. 定期备份：Notebook 定期导出备份
  5. 代码格式化：保存时自动格式化
  6. 版本控制：所有代码（包括 SQL）都纳入 Git
  7. 敏感信息：数据库密码等不要硬编码

六、练习

  1. 安装 VS Code，配置 Python 开发环境
  2. 安装 Jupyter，创建一个数据分析 Notebook
  3. 安装 DataGrip，连接本地或远程数据库
  4. 在 VS Code 中安装并配置常用插件
  5. 配置 Settings Sync，同步开发环境
  6. 学习并使用每个工具的快捷键
  7. 创建一个包含代码、文本、图表的完整 Notebook
