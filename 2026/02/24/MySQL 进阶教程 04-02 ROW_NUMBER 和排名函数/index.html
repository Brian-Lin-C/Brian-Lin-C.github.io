MySQL 进阶教程 04-02 ROW_NUMBER 和排名函数

在窗口函数中，排名函数是最常用、最重要的一类函数。本文将详细讲解 ROW_NUMBER、RANK、DENSE_RANK 三种排名函数的区别和使用方法。

一、排名函数简介

MySQL 提供三种主要的排名函数：

函数 说明 排名序列示例
ROW_NUMBER() 连续排名，无重复，不跳过 1, 2, 3, 4, 5
RANK() 跳跃排名，有重复，跳过 1, 2, 2, 4, 5
DENSE_RANK() 密集排名，有重复，不跳过 1, 2, 2, 3, 4

6.1 三种函数对比表

对比项 ROW_NUMBER RANK DENSE_RANK

成绩/比赛排名 DENSE_RANK 考试排名、比赛名次
允许跳过的排名 RANK 体育比赛（并列冠军，无亚军）

八、小结

核心知识点

  1. ROW_NUMBER()

    • 连续编号，不考虑并列
    • 适用于 Top N、去重、分组取最新

  2. RANK()

    • 并列同名，后续排名跳过
    • 适用于允许跳过的排名场景

  3. DENSE_RANK()

    • 并列同名，后续排名不跳过
    • 适用于需要连续排名的场景

  4. PARTITION BY

    • 用于分组内排名
    • 每个分区独立计算排名

  5. ORDER BY

    • 决定排名的排序方向
    • 可以多列排序
